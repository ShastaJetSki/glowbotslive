Stripe / Payment
💳 Connect Stripe for Automatic Billing
Connect Stripe to automatically charge the account owner each month. Collaborators on the free billing model (those with their own plan) are never charged. New paid collaborators are added to the subscription immediately on invite acceptance.
Stripe Setup Guide
STEP 1 — Add Supabase Secrets

Go to Supabase Dashboard → Settings → Edge Functions → Secrets and add:

STRIPE_SECRET_KEY=sk_live_YOUR_KEY_HERE
STRIPE_WEBHOOK_SECRET=whsec_YOUR_WEBHOOK_SECRET
STRIPE_COLLAB_PRICE_ID=price_YOUR_COLLAB_PRICE_ID
STEP 2 — Run This SQL in Supabase
ALTER TABLE tenants
ADD COLUMN IF NOT EXISTS stripe_customer_id text,
ADD COLUMN IF NOT EXISTS stripe_subscription_id text,
ADD COLUMN IF NOT EXISTS stripe_price_id text,
ADD COLUMN IF NOT EXISTS stripe_collab_price_id text,
ADD COLUMN IF NOT EXISTS stripe_status text DEFAULT 'inactive',
ADD COLUMN IF NOT EXISTS next_billing_date date;

CREATE TABLE IF NOT EXISTS billing_invoices (
  invoice_id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  tenant_id uuid REFERENCES tenants(tenant_id) ON DELETE CASCADE,
  stripe_invoice_id text UNIQUE,
  amount_cents int,
  status text,
  billing_period_start date,
  billing_period_end date,
  paid_at timestamptz,
  created_at timestamptz DEFAULT now()
);
STEP 3 — Create Stripe Products

In your Stripe Dashboard → Products, create:

Product 1: "EventLogicPro Base Plan" — $19.95/month recurring

Product 2: "EventLogicPro Collaborator Seat" — $5.00/month per seat

Copy both Price IDs (start with price_) and paste them into the GlowBots Admin → Tiers tab for the matching tier.

STEP 4 — Add Webhook in Stripe

In Stripe → Developers → Webhooks, add endpoint:

https://kxqkwpkmtgvmthpafoqx.supabase.co/functions/v1/stripe-webhook

Enable these events: invoice.paid · invoice.payment_failed · customer.subscription.updated · customer.subscription.deleted
